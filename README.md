# 双轮平衡车运动控制系统
## 硬件部分
* MCU:STM32F103C8T6
* 姿态采集:MPU6050
* 电机驱动:TB6612FNG

## 开发平台
* STM32CubeMX
* Keil MDK5

## 软件算法
* 卡尔曼滤波
* 串级PID控制
* 速度一阶低通滤波
